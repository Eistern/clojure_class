clazz.generic

call

(call name params)

Call all suitable generic function implementations for the passed parameters params with types. The invocation order is determined by the fact how close ‘params’ types to the particular generic function implementation

declare_generic!

(declare_generic! name params_list)

Declares a generic function with a name and a list of parameters in params_list. You can’t declare two generic functions with the same name.

declare_method!

(declare_method! name additional_params params function)

Declare generic function implementation with name, params parameters with types and function which will be executed with this implementation

generic_method_exists?

(generic_method_exists? name params)

Checks if generic method with name and params is declared.

get_generic_functions

(get_generic_functions)

Returns declared generic functions (name and parameters). Changes to this object will not affect generic context

get_generic_impls_map

(get_generic_impls_map)

Returns declared generic functions implementations (name and parameters with types). Changes to this object will not affect generic context

get_keys_from_list

(get_keys_from_list params)

Get list of parameters names from list pf parameter names with types.

get_metric

(get_metric method_params passed_params)

Get Long metric between two lists of parameters. method_params are parameters of one of a declared generic functions implementations. passed_params are parameters passed with generic function invocation.

get_metric_for_params

(get_metric_for_params params method_impls)

Get list of pairs with metric between params and method implementation and method implementation itself for every implementation in method_impls

get_param_by_name

(get_param_by_name name params)

Get param object by its name

params_coincided?

(params_coincided? passed_params actual_params)

Checks if two parameters names lists are the same.

search_class

(search_class current_name target_name)

Returns Long distance in inheritors tree from the target_name class to the ‘current_name’ class. Returns -1 if ‘current_name’ is not the base class for the target_name

search_class_in_classes_list

(search_class_in_classes_list classes_list target_name)

Search class with a name the target_name among the list of classes classes_list. Returns Long distance in inheritors tree from the target_name class to the one of classes from classes_list, which is base class for target_name. Returns -1 if there is no base class for the target_name class among the classes_list
